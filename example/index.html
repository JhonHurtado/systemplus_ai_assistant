SystemPlus AI Assistant

Ejemplo de Integración

Esta página demuestra cómo integrar el asistente virtual de SystemPlus en cualquier sitio web. El widget se carga mediante un simple script tag con opciones de configuración personalizables.

Código de Integración

Agregue el siguiente código en el archivo HTML de su sitio:

<script id="systemplus-widget-script" src="https://asistente.systemplus.systems/dist/widget.min.js" data-config='{ "color": "#be3124", "title": "Asistente System Plus", "initialMessage": "¡Hola! Soy el asistente virtual de System Plus. ¿En qué puedo ayudarte?", "theme": "light", "avatarUrl": "https://systemplus.systems/images/svg/icon_bot.svg", "apiBasePath": "https://asistente.systemplus.systems", "position": "bottom-right", "fontFamily": "Poppins, sans-serif", "enableSound": true, "suggestionsEnabled": true, "size": "medium", "autoOpen": false }' async> </script>

Opciones de Personalización

El widget se puede personalizar modificando el objeto JSON en el atributo data-config. Estas son algunas opciones disponibles:

  • color: Color principal del widget (código hexadecimal)
  • title: Título del widget
  • theme: Tema - "light" o "dark"
  • position: Posición - "bottom-right", "bottom-left", "top-right" o "top-left"
  • size: Tamaño - "small", "medium" o "large"
